# Validation Worksheets

Use these worksheets to record review decisions. They can be copied into a spreadsheet.

Decision labels:

- `approve`
- `approve_with_note`
- `rewrite`
- `reject`
- `needs_more_evidence`
- `outside_my_expertise`

## 1. System-Level Review
