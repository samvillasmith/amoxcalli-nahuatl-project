# Validation Request

## What We Need

We are asking reviewers to validate **Modern Standard Nahuatl (MSN)** as a usable written standard.

The intended outcome is a public learner book called:

```text
Learn Modern Standard Nahuatl
```

That book should teach someone enough MSN to:

- read basic MSN prose
- write simple paragraphs
- write a short book in MSN
- understand register labels
- avoid confusing MSN with classical citation forms
- recognize when a text is using elevated poetic/literary style
- eventually compose poetry in a properly marked MSN poetic register

## What Reviewers Should Judge

Please review the system, not just isolated examples.

We need decisions on:

- whether the MSN orthography is teachable and acceptable
- whether the register distinctions are clear and honest
- whether core grammar patterns are correct enough to teach
- whether core vocabulary forms are acceptable as headwords
- whether example sentences are understandable and natural enough
- whether the proposed learner-book sequence makes pedagogical sense
- whether the system can support prose and poetry without blurring source registers

## Review Decision Labels

Use these decision labels:

- `approve`
- `approve_with_note`
- `rewrite`
- `reject`
- `needs_more_evidence`
- `outside_my_expertise`

## What Counts As Valid

An item is valid only when it has:

- a clear source basis
- a clear register label
- an acceptable written form
- a reviewer decision
- notes for any limitation

If something is useful but not yet public-ready, mark it as `approve_with_note` or `needs_more_evidence`.

## What We Do Not Want

Please do not silently normalize everything into classical Nahuatl.

Please do not approve a form just because it appears somewhere in a source. We need to know whether it belongs in **MSN**, what register it belongs to, and whether a learner can trust it.
