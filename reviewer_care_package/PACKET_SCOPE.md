# Packet Scope

This package validates **MSN as a system**.

It is not a request to validate one isolated draft.

## In Scope

- MSN neutral prose standard
- MSN public prose standard
- MSN poetic/literary extension
- register boundaries
- orthography
- source hierarchy
- core grammar
- core lexicon
- example sentence policy
- learner-book sequence
- ability to write books
- ability to compose poetry

## Out Of Scope For This Packet

- final publication layout
- marketing copy
- app UI review
- exhaustive database row-by-row review in a single pass
- approving every classical citation as modern usage

## How To Handle Examples

Examples are evidence tests. They are not the whole target.

If an example reveals a problem, mark the underlying system issue:

- spelling rule problem
- grammar rule problem
- register label problem
- lexical choice problem
- source evidence problem
- pedagogy problem

## Full Validation Goal

The system is ready only when reviewers can say:

```text
This standard is clear enough and valid enough to teach in Learn Modern Standard Nahuatl,
and learners can use it to write prose, books, and poetry with responsible register labels.
```
