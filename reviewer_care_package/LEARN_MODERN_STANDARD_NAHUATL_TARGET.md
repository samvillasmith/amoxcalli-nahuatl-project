# Target Book: Learn Modern Standard Nahuatl

The final goal is a learner book:

```text
Learn Modern Standard Nahuatl
```

This book should be the practical proof that MSN works.

## Reader Outcome

After using the book, a learner should be able to:

- read basic MSN prose
- understand the spelling system
- understand the difference between spoken EHN, MSN neutral, MSN public, MSN poetic, and classical citation
- write simple sentences
- write short paragraphs
- create a short book in MSN
- recognize when a word or phrase is not yet validated
- begin composing poetry with proper register awareness

## Proposed Book Structure

1. What Modern Standard Nahuatl Is
2. Sounds, Spelling, And Reading MSN
3. Nouns And Basic Word Forms
4. Simple Sentences
5. Questions
6. Negation
7. Possession
8. Verbs, Person, And Aspect
9. Location, Movement, And Time
10. Description
11. Food, Home, School, And Community
12. Reading Short MSN Texts
13. Writing Paragraphs
14. Writing A Short Book
15. Register And Style
16. Introduction To Poetic/Literary MSN
17. Source Notes, Glossaries, And Responsible Use

## Validation Questions For The Book

- Is this sequence teachable?
- What grammar is missing?
- What examples are unsafe or unnatural?
- What vocabulary must be validated before publication?
- Where should spoken EHN be introduced?
- Where should classical citation be introduced?
- How should poetry be introduced without confusing it with neutral prose?
- What must a learner know before writing a book or poem?

## Completion Standard

The book is not complete until the underlying MSN system has been reviewed at the level of:

- orthography
- register labels
- core grammar
- core vocabulary
- examples
- prose-writing patterns
- poetic/literary extension
- source and validation notes
