# Start Here

This packet asks for full validation of **Modern Standard Nahuatl (MSN)** as a written standard.

The target is not approval of one draft or one example text. The target is enough validated MSN to support a public learner book called:

```text
Learn Modern Standard Nahuatl
```

That book should teach a learner to read MSN, write paragraphs, write short books or prose, understand register labels, and begin poetry or literary composition without confusing MSN neutral, MSN public, MSN poetic, spoken EHN, and classical citation material.

## What Counts As A Real Review

A useful review answers these questions:

- Is the MSN writing standard acceptable and teachable?
- Are the register boundaries honest and clear?
- Are core grammar patterns correct enough for learners?
- Are core headwords and example sentences acceptable?
- Can this system support original prose and books?
- Can it support poetry and literary/devotional writing with proper safeguards?
- What must be rewritten, rejected, or held for more evidence?

## Output We Need Back

Please return one of the following:

- completed `VALIDATION_WORKSHEETS.md`
- a spreadsheet using the same columns
- annotated notes with clear `approve`, `approve_with_note`, `rewrite`, `reject`, `needs_more_evidence`, or `outside_my_expertise` labels

The most important thing is not long commentary. The most important thing is clear decisions with better forms or examples where something should be changed.
